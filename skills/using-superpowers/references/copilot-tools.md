# Copilot CLI Tool Mapping

| Claude Code | Copilot CLI |
|-------------|-------------|
| `Skill` tool | Native skill discovery from plugins |

| `Write` | Native file writing |
| `Edit` | Native file editing |
| `Bash` | Native shell execution |
| `TodoWrite` | `todowrite` |
